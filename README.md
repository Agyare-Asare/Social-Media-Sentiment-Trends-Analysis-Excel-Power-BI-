# Social Media Sentiment Trends Analysis (Excel & Power BI)
## Project Overview
Social media platforms generate huge amounts of user opinions every day. This project explores how sentiment varies across different social media platforms and posting times, with the aim of identifying overall emotional trends and behavioral patterns.
Using labeled sentiment data from Twitter, Facebook, and Instagram, the analysis focuses on understanding when users are more likely to post positive, neutral, or negative content and how sentiment differs by platform and time of day.
The final output is an interactive Power BI dashboard that allows users to explore sentiment patterns over time.
## Dataset
The dataset contains social media posts with pre-labeled sentiment values.
Key fields include:
Year
Month
Day
Time of Tweet (morning, noon, night)
Text
Sentiment (positive, neutral, negative)
## Platform (Twitter, Facebook, Instagram)
All data is anonymized and contains no personally identifiable information.

## Analysis Approach
1. Data Preparation (Excel)
Combined date fields into a usable date format
Standardized categorical values such as platform and time of tweet
Prepared the dataset for analysis and visualization in Power BI
2. Exploratory Analysis & Visualization (Power BI)
Analyzed overall sentiment distribution
Compared sentiment across different social media platforms
Examined sentiment patterns by time of day
Tracked sentiment trends over multiple years using interactive filters
## Key Insights
Neutral sentiment is the most common across all platforms daily
Instagram shows a higher proportion of positive sentiment compared to Twitter and Facebook
Negative sentiment appears more frequently during night-time posting hours
Sentiment patterns change over time, indicating shifts in online user behavior
